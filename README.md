## README – EduTrack Student Management System

EduTrack is a Student Management System developed using Java, JDBC, and MySQL. The purpose of this project is to manage student records efficiently through a menu-driven console application. The system allows users to perform various operations such as adding new students, viewing all student records, searching for a particular student, updating student details, deleting student records, and exporting student data to a text file. This project demonstrates the practical implementation of Java programming concepts, database connectivity using JDBC, SQL operations, file handling, and modular programming.

The project uses MySQL as the backend database. The database named **students_schema** contains a table called **students**, which stores information such as Student ID, Name, Course, Semester, Grade, and Email. The Student ID serves as the primary key, ensuring that each student record is unique. JDBC (Java Database Connectivity) is used to establish a connection between the Java application and the MySQL database, allowing data to be inserted, retrieved, updated, and deleted.

The project is organized into multiple Java classes to improve readability and maintainability. The **DBConnection** class is responsible for establishing a connection with the MySQL database. The **MainMenu** class acts as the entry point of the application and displays the available options to the user. The **AddStudent** class handles the insertion of new student records into the database. The **ViewStudents** class retrieves and displays all records from the database. The **SearchStudent** class allows users to find a student using their Student ID. The **UpdateStudent** class enables modification of student information such as Name, Course, Semester, Grade, and Email, while preventing changes to the Student ID. The **DeleteStudent** class removes student records from the database, and the **ExportData** class exports all student records into a text file for reporting and backup purposes.

To run the project, the MySQL server must be running, and the required database and table should be created beforehand. The MySQL JDBC Connector JAR file must be added to the Eclipse project's build path to enable database connectivity. After configuring the database URL, username, and password in the DBConnection class, the MainMenu class can be executed to start the application.

This project provides hands-on experience in Java programming, object-oriented design, database management, SQL query execution, exception handling, and file operations. It can be further enhanced by adding features such as a graphical user interface using Java Swing, login authentication, attendance management, PDF report generation, and advanced student analytics. Overall, EduTrack serves as a practical example of integrating Java applications with relational databases to build a complete student information management solution.
